# Bloc

## Prerequisites
1. Node.js

## Installation && Run
1. Download source code, put it into folder `$Bloc$`.
2. Install `Node.js`.
3. Open terminal/CMD, go to folder `$Bloc$`.
4. Server:
    1. Go to folder `$Bloc$/smart-brain-api-master/`.
    2. Run `npm install`.
    3. Run `npm start`.
5. Client:
    1. Go to folder `$Bloc$/meetup/src/`.
    2. Run `npm install`.
    3. Run `npm start`.
